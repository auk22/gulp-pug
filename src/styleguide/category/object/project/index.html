Carousell

/sass\object\project\_carousell.scss
  • project
  • library

スライダーコンポーネント。
外部のライブラリswiperを利用しています。
_carousell.scss以外にも後述のファイルが必要です。

  • ・foundation/lib/_swiper.scss
  • ・js/swiper.min.js

swiperの設定はcommon.jsで行います。

※p-carousellの子要素はswiperの命名に従います。 http://idangero.us/swiper/#.WTd7MuvyiHs

デモグラフィックが明確な独自のオーディエンスデータを有効活用し、精密で効率の良い広告配信を実現した広告プラットフォームサービスです。

        <img src="img/img_slide01.jpg" alt="">
      </div>
      <div class="swiper-slide">
        <img src="img/img_slide02.jpg" alt="">
      </div>
      <div class="swiper-slide">
        <img src="img/img_slide03.jpg" alt="">

  </div><!-- /.swipter-container -->
